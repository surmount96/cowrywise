# cowrywise test

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Customize configuration for .env
Add the following to the file

```
VUE_APP_BASE_URL={url}
VUE_APP_TOKEN={Access_token}
```
